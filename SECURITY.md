# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 6.x.x   | :white_check_mark: |
| < 6.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities to us via the built-in [GitHub Advisories](https://github.com/schibsted/sum/security/advisories/new). 

We will strive to respond to your report as quickly as possible. To protect our users, we kindly ask that you allow the vulnerability to be fixed before disclosing it publicly.
